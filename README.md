Meta Ad Performance Dashboard

📌 Project Overview

The Meta Ad Performance Dashboard is an interactive analytics dashboard built to monitor and analyze Facebook and Instagram advertising campaigns. It provides a clear overview of campaign performance through KPIs, charts, and filters, helping marketers make data-driven decisions.

🎯 Objectives

Monitor advertising performance in one place.

Track engagement and conversion metrics.

Analyze audience behavior by age, gender, country, and time.

Improve campaign effectiveness using data insights.

📈 Dashboard Insights


Generated 123.8K impressions with 14.7K clicks.

Overall CTR is 11.86%, indicating strong user interaction.

Engagement Rate is 13.60%, showing high audience engagement.

Total 708 purchases were recorded.

Conversion Rate is 4.82%.

Female users contributed the highest percentage of comments.

Most engagement comes from users aged 20–30 years.

Weekly comments fluctuate, showing peak engagement on selected days.

Audience engagement is distributed across multiple countries.

Comment activity changes throughout the day, helping identify the best posting hours.
